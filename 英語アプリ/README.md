# Song Pronunciation Coach

英語の歌詞をローカルで発音チェックするための、依存なしのシンプルなブラウザアプリです。

## できること

- 練習したい英語の歌詞やフレーズを入力
- マイクで録音しながら英語音声認識
- 認識結果と見本歌詞を単語ごとに比較
- 一致率のざっくり確認
- 自分の録音をその場で再生

## 使い方

1. ターミナルでこのフォルダに移動します。
2. 次のコマンドでローカルサーバーを起動します。

```bash
python3 -m http.server 4173
```

3. ブラウザで [http://localhost:4173](http://localhost:4173) を開きます。
4. マイク利用を許可して、歌詞を入れてから `録音してチェック開始` を押します。

## ファイル

- `index.html`: 画面
- `styles.css`: 見た目
- `app.js`: 録音、音声認識、比較ロジック

## メモ

- `SpeechRecognition` / `webkitSpeechRecognition` を使うため、Chrome 系ブラウザがいちばん試しやすいです。
- これは「発音の絶対評価」ではなく、「認識された単語が見本とどれくらい近いか」を見る最初の土台です。
- あとから改善しやすいように、外部依存を入れずに作っています。

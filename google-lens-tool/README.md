# Lens Desk

Google Lensのように、画像を見ながら検索キーワードを整理するための小さなWebアプリです。

## 使い方

`index.html` をブラウザで開きます。

## 花の名前付き保存

1. 花の写真を選びます。
2. Google Lensで花の名前を調べます。
3. 花の名前を入力します。
4. 保存画像を作ります。
5. PNGで保存します。

- 画像を選ぶとプレビューします。
- 「読む」で画像内の文字をOCRします。
- 文字が小さい時は、画像上をドラッグして範囲選択してから読むと精度が上がります。
- 「名前付き保存」で、調べた物体名を入れたPNG画像を作成して保存できます。
- 画像上をドラッグして範囲選択すると、その物体部分だけを名前付き画像にできます。
- 対応ブラウザではQRコードやバーコードを検出します。
- 主要な色をクリックすると検索キーワードに追加できます。
- Google検索、画像検索、Google Lensアップロード画面へ移動できます。

## 注意

Google Lensの内部解析APIを直接使うものではありません。OCRにはTesseract.jsを使っているため、初回利用時はインターネット接続が必要です。商品や植物などの物体名推定はGoogle Lens本体ほど高精度にはできません。
